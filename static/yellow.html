Static Pokémon are encountered at fixed overworld locations — not in wild grass or trainer parties. Examples: Articuno, Zapdos, Moltres, Mewtwo, Snorlax, Lapras, and fossil revives (Omanyte, Kabuto, Aerodactyl).

Quick Start
1

Get Yellow Legacy

Download the Pokemon Yellow Legacy source from GitHub. Extract the ZIP — the folder contains a Makefile at its root.

2

Set Source & Output

Click Browse… next to Source Directory and select the extracted Yellow Legacy folder. The output path auto-fills; change it if you like.

3

Configure Options

Choose your randomization settings across the tabs. A random seed is generated automatically — enter a specific number to reproduce results.

4

Randomize (+ optional ROM build)

Click Randomize!. With "Build ROM" checked, the tool calls make in the output folder and produces pokeyellow.gbc. You'll be asked where to save the ROM.

Requirements
Building the ROM requires RGBDS (the Game Boy assembler) and GNU Make. Install RGBDS from github.com/gbdev/rgbds/releases and Xcode Command Line Tools (xcode-select --install) for Make. Yellow Legacy is known to work with RGBDS 0.5.x and 0.6.x. Check the project README for the recommended version.
What Gets Randomized

Wild encounters — grass, water, and fishing (Old Rod, Good Rod, Super Rod) Pokémon in every map.
Trainers — all trainer parties (species only; levels preserved).
Static Pokémon — fixed overworld encounters: legendary birds, Mewtwo, Snorlax, Lapras, fossil revives, and gifts.
Starter — the single starter Pokémon (replaces Pikachu).
Trades — NPC in-game trade species.
Items — overworld field items (visible Pokéballs and hidden items); configure starting bag and PC items for new games.
General — easier evolutions and full HM compatibility.
