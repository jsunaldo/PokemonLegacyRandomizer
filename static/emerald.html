Pokémon Pool

⚠️ At least one generation must be selected.

Starter Pokémon

Emerald Legacy has three starters: Treecko, Torchic, and Mudkip. The randomizer patches the starter selection array in src/starter_choose.c.

Pokémon

Static Pokémon are encountered at fixed overworld locations — not in wild grass or trainer parties. Examples: Kyogre, Groudon, Rayquaza, the Regis, Latias/Latios, and gift Pokémon such as the Castform, Beldum, and the fossil revivals.

ℹ️ The Pokémon pool is controlled by the General tab.
Only encounters using the setwildbattle script command are detected. Most major static encounters in Emerald use object-event battles and will not be affected by this setting.

Pokémon

In-game trades are NPC exchanges built into the game — e.g. giving a Ralts and receiving a Bagon. The parser detects trades from src/data/trades.h (or equivalent trade data in the source tree).

Sub-options

Field Items

Field items are items found in the overworld — visible Pokéball sprites on the ground and hidden items found with the Itemfinder. Key items and HMs are never replaced.

Sub-options

Configure what items are in your bag and PC at the start of a new game. Only takes effect on a new game — not a continued save.

Steps
1

Clone the Emerald Legacy source

Download the Pokemon Emerald Legacy repository and place it in a convenient folder.

git clone https://github.com/cRz-Shadows/Pokemon_Emerald_Legacy.git

2

Set the Source & Output directories

Use the Browse buttons at the top to set:

  • Source — the cloned repository folder
  • Output — a new empty folder where the patched source will be written

Source and Output must be different directories — the source is never modified.
3

Configure and randomize

Set your options on each tab, then click Randomize! on the Randomize tab. The randomizer will copy the source, patch the relevant files, and (if Build ROM is checked) compile a .gba file.

4

Load in an emulator

Open the generated EmeraldLegacy_Randomized_SEED.gba in any GBA emulator (mGBA, RetroArch, etc.).

New game required: Starting items and PC Pokémon only appear when you start a new game. Delete your save or use a fresh emulator state before testing.
Tips
  • Save your settings — use Save Settings on the Randomize tab to export a JSON file you can load back later.
  • Reproducible runs — save both settings and seed with Save Settings + Seed. Sharing that file recreates the exact same randomized ROM.
  • Full HM Compat — strongly recommended for randomized runs. Without it, a randomized party may be unable to learn required HMs.
  • Easier Trade Evolutions — also recommended. Without it, trade-evolving Pokémon (Kadabra → Alakazam, etc.) can only be obtained pre-evolved.
  • Wild data is stored in a single JSON file (src/data/wild_encounters.json), making randomization fast and reliable.
